# SH3 Apps

## About

Monorepo for SH3 official applications.

## Apps

- [sh3-diagnostic](docs/sh3-diagnostic.md) — Diagnostic tool used to debug/develop SH3 apps
- [sh3-editor](docs/sh3-editor.md) — Reusable text editor shard for SH3
- [sh3-mock](docs/sh3-mock.md) — SH3 mock application
- [sh3-registry](docs/sh3-registry.md) — Server shard for hosting and managing a package registry
- [sh3-style](docs/sh3-style.md) — Theme editor shard for SH3
