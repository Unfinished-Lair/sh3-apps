# sh3-connector-r2

Back up SH3 documents to a user-owned Cloudflare R2 bucket. v0.1.0 is manual:
users pick files/folders to back up from the file-explorer, or run a
whole-scope backup from the R2 Backup app. Import is cherry-pick from the
R2 bucket back into the local tenant.

See `docs/superpowers/specs/2026-04-20-sh3-connector-r2-design.md` for the
full design.
