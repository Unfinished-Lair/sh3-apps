# sh3-editor-fiddle

SH3 fiddle apps for testing the `sh3-editor` shard's surfaces in isolation.

v1 ships one app: **Inspector Fiddle** — type a JSON value and an `InspectorMeta` JSON, see the inspector render the result.

User-facing reference: [`docs/sh3-editor-fiddle/inspector-fiddle.md`](../../docs/sh3-editor-fiddle/inspector-fiddle.md).
